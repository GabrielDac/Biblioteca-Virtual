ℹ️ Acerca de esta herramienta

Biblioteca Virtual permite organizar y consultar libros digitales como en una estantería real: cada documento se describe en una ficha catalográfica, recibe su clasificación CDU y su signatura topográfica, y se ubica en el estante como un lomo con tejuelo, con grosor y altura proporcionales a sus páginas y formato. Los documentos se vinculan mediante enlaces (Drive u otros servicios) en la versión web, o seleccionando archivos locales en la aplicación de escritorio, disponible para Windows, macOS y Linux. Técnicamente es una aplicación construida en HTML, CSS y JavaScript puros, sin servidor de datos: el catálogo se guarda en el navegador de cada usuario, y puede exportarse e importarse como archivo JSON para respaldo o traslado. La versión de escritorio empaqueta la misma interfaz con el framework Tauri (Rust), y sus instaladores se compilan automáticamente para los tres sistemas operativos mediante integración continua. Próximamente se podrá clasificar también por Dewey. Software libre bajo licencia GNU GPL-3.0.

Otras herramientas: Clasificación Documental · Referencias Bibliográficas (los servidores de Render pueden demorar unos 30 segundos en la primera carga)

⬇ Descargar la aplicación

La misma biblioteca, instalada en tu computadora: abre los PDF locales con un clic, funciona sin internet y elegís los archivos con el explorador, sin copiar rutas.

Mac (.dmg) Windows (.exe) Linux (.AppImage)

Otras opciones — Linux: .deb · .rpm  ·  Windows: .msi

Al abrir por primera vez, Windows y Mac pueden mostrar un aviso de «desarrollador no identificado»: es lo esperable en software libre sin certificado comercial. En Mac: clic derecho sobre la aplicación → Abrir → Abrir (solo la primera vez).
